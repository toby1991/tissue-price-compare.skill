# tissue-price-compare

抽纸 / 纸巾比价 AgentSkill，用来判断一组抽纸参数是否比 User 当前保存的最佳购入基准更划算。

## 用途

当用户给出抽纸规格和价格时，快速回答：

- 是否划算 / 是否值得买
- 单包价、单抽价是多少
- 比当前基准便宜或贵多少
- 按当前基准折算，合理价大约是多少
- 层数、抽数、包数、尺寸、品牌是否存在不可直接比较的问题

典型触发问题：

```text
3层 110抽 24包 29.9，划算吗？
抽纸 3层 150抽 40包 69.91 能买吗？
这款纸巾 4层 100抽 30包 45 元怎么样？
```

## 当前基准

User 当前已购最划算型号：

- 价格：¥29.34 / 箱
- 数量：24 包
- 规格：3 层，110 抽 / 包
- 单包价：¥1.2225 / 包
- 单抽价：约 ¥0.01111 / 抽

同规格下，低于这个单抽价才算真正划算。

## 判断规则

核心比较口径是「单抽价」，但不能忽略质量维度。

计算方式：

```text
单包价 = 总价 / 总包数
单抽价 = 总价 / 总包数 / 每包抽数
合理价 = 基准单抽价 × 总包数 × 每包抽数
差异百分比 = (当前单抽价 / 基准单抽价 - 1) × 100%
```

默认结论：

- 低于基准：划算 / 可以买
- 贵 0% ~ 3%：小幅偏贵，只有尺寸或品牌更好才买
- 贵超过 3%：不划算，跳过
- 便宜超过 5%：明显划算
- 便宜超过 15%：非常划算，优先买，但仍需注意尺寸/品牌缩水

如果层数不是 3 层，必须明确提示「不是同规格对比」。

## 脚本用法

确定性计算器在：

```bash
skills/tissue-price-compare/scripts/compare_tissue.py
```

单个商品：

```bash
python3 skills/tissue-price-compare/scripts/compare_tissue.py \
  --layers 3 \
  --draws 150 \
  --packs 40 \
  --price 69.91
```

多个商品：

```bash
python3 skills/tissue-price-compare/scripts/compare_tissue.py \
  --item "3,110,24,29.34,当前基准" \
  --item "3,150,40,69.91,候选款"
```

`--item` 格式：

```text
layers,draws,packs,price[,name]
```

## 推荐回复格式

Discord 里不要用表格，直接短句输出：

```text
不划算。

这款：69.91 ÷ 40 ÷ 150 = ¥0.01165/抽
基准：¥0.01111/抽

比基准贵约 4.8%。按基准折算，合理价约 ¥66.68；除非尺寸/品牌明显更好，否则跳过。
```

有歧义时先问清楚，或直接列两个场景：

```text
这里有歧义：39.98 是 3 箱合计价，还是每箱单价？
如果是合计价：...
如果是单箱价：...
```

## 维护基准

如果 User 明确更新新的最佳购入价，需要同步修改：

1. `SKILL.md` 里的 Baseline
2. `scripts/compare_tissue.py` 里的 `BASELINE_*` 常量
3. 相关长期记忆 / 当日 memory 记录
4. 如果需要分发，再重新打包 `.skill`

不要只改文档不改脚本，否则判断会漂。
